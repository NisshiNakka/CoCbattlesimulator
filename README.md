# README
## サービス概要
どんなサービスなのかを３行で説明してください。


このアプリは、TRPG(*1),特にCoCにおいての戦闘のシミュレーションを行うことができるアプリです。調整したデータはそのままココフォリア(*2)に出力することが可能であり、テストをしたデータをスムーズに本番で使うことができます。

(* TRPG ... テーブルトークRPGの略称。ダイスなどを使用して進めるアナログゲーム )
(* ココフォリア ... TRPGを行うための環境を作ることができる現在主流のサイト。)

## このサービスへの思い・作りたい理由
このサービスの題材となるものに関してのエピソードがあれば詳しく教えてください。
このサービスを思いつくにあたって元となる思いがあれば詳しく教えてください。


1.自分や友人が実際にTRPGのシナリオを作成していた時に、ゲーム内の戦闘バランス調整を行う際、多くの手間がかかったことから、この作業を短縮することはできないかと感じたことが理由の一つ。

2.創作を行うことが好きで、「創ること」による楽しさをもっと感じてほしいと考えているため、
「創作をもっとしやすく」することができるサービスを作りたいと思ったのが理由。

## サービスの利用イメージ
ユーザーがこのサービスをどのように利用できて、それによってどんな価値を得られるかを簡単に説明してください。

ユーザーはこのサービスにアクセスし、敵・味方のステータスなどの値を入力します。
その後、条件を決めて、実行ボタンを押すことで結果が表示されます。
これにより、自分が作ったシナリオの戦闘バランスを考え、テストを行うことが簡単になります。

## サービスの差別化ポイント・推しポイント
似たようなサービスが存在する場合、そのサービスとの明確な差別化ポイントとその差別化ポイントのどこが優れているのか教えてください。
独自性の強いサービスの場合、このサービスの推しとなるポイントを教えてください。

このサービスのポイント
1.独自性が高い　他に代替できそうなアプリはない
2.ココフォリアの場合、すべての判定を一回一回入力してテストを行わなければならない。
弊アプリの場合、ステータスや条件を決めたらあとは１クリックで結果がわかるようになり、テスト時間が短縮される。
